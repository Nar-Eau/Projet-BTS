2 Ma Partie qui consiste a :
	- Choisir un support pour créer l'application de création de sondages.

L'application de création de sondages comportera une IHM avec :
	- Une communication avec la base de données

	- Des champs pour créer les sondages qui seront stockés dans la bdd.
	( Dans ces champs on définira la question, les deux options, ainsi qu'une image pour chaque option.
	 La date de début et la date de fin du sondage. )

	- Champs pour accéder aux précédents sondages, avec possibilité de les modifier ou/et les supprimer.
	( Les champs recupèreront les anciens sondages stockés dans la Bdd.)

3 Réalisation du projet
